How Do Websites Work?

The internet works through a simple request and response process from client to server.

Quick Terms

Client
your computer
A computer that is recieving information from a server.
Server
the computers with the website
the computer that sends information to clients

How does this process work?

First you need to get software: a browser. Then—through an ISP —your browser will connect to a DNS server via a HTTP. The DNS server will then—through HTTP—respond by providing the IP address of the site you're looking to visit. The DNS is pretty analogous to a phonebook. From there your computer will be able to connect to the web server where the browser can then request and recieve the web page.

Computer information to visuals

The computer needs instructions to display the webpage to the user. To do that front-end developers, the people who make webpages, use a markup language. These markup languages use tags (markup) to provide structure and meaning to the webpages. Our internet overlords, W3C , have graced us with many standards over the time of the web; the most recent standard is HTML5. HTML is not the only part of a website. CSS styles the presentation and appearance of a site, while Javascript provides dynamics.
